1.0.0 Potential Drug-drug Interaction Clincal Decision Support Implementation Guide

1.1.0 Introduction

This implementation guide for potential drug-drug interaction (PDDI) clinical decision support (CDS) specifies both a knowledge representation format for PDDI logic and CDS services for PDDI with electronic health record (EHR) systems. Specifically, the implementation guide includes specifications for:

  1. How to represent PDDI logic in CQL and evidence in computational format using the FHIR Clinical Reasoning module.
  2. How to use CDS Hooks as a mechanism for EHRs to request PDDI CDS from CDS services.
